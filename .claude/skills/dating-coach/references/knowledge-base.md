# 两性知识库

按主题组织，供「流程 B：知识问答」按需查阅。回答用户时**结合他的具体处境**重组这些知识，不要整段照搬。

> 这些方法源自恋爱教学流派，取其沟通/心态/形象的可借鉴部分。运用时始终守 SKILL.md 的底线：真诚、尊重、自我成长为根。

---

## 0. 底层逻辑（一切技巧的地基）

- **价值 + 可得性 = 吸引**。光有价值不够，要让她觉得"够得着"；但也不能毫无门槛。
- **不暴露需求感**是第一原则，需求感是吸引的最大杀手。区分"需求"和"需求感"——可以想要，但不能急、不能黏、不能有求必应。
- **女人慕强；追求者价值低于选择者**。把自己放在"被选择"的位置，而非一味追逐讨好。
- **她因为喜欢你才天天和你聊，不会因为你天天找她聊而喜欢你**。别用聊天频率消耗她的耐心。
- **经营自己 > 死追一人**。她靠近你，是因为你状态好、有溢出的养分。"与其追马，不如种草；花香蝶自来。"
- **数量改变质量**。选择太少 → 把唯一对象当全世界 → 患得患失、降低底线。要扩圈。
- **配得感**取代"付出—回报"交易思维。

## 1. 认识 / 搭讪

- **扩大社交面**：兴趣社群、运动、朋友局、同事同学、社交软件。机会来自基数。
- **第一印象 = 形象 + 状态**（见第 6 节）。先把硬件和精神状态拉起来。
- **破冰**：率先打破陌生感，下意识不把对方当陌生人。开场一个表情/一句轻松的话即可，别正式"你好"也别马上查户口。切入点：评她的外形（"小可爱/大眼睛"）、从她朋友圈或当下情境找话题、描述自己的状态制造熟人感。
- **搭讪心态**：高姿态、不猥琐、被拒无妨。引发情绪波动就算成功。
- **要联系方式**：在互动有来有回、她被你逗笑或主动接话时自然要；可用"时间桥梁"埋下一次见面的预期（"我朋友周末有个局，你来一起玩"）。

## 2. 聊天

### 总原则
- **聊天的核心是传递情绪**，没有情绪只会消耗耐心。"无效聊天一年，不如有效聊天一天。"
- **男性思维解决问题，女性思维要情绪共鸣**。她倾诉时先共情，别急着给方案。
- 节奏公式：**聊她 → 聊我 → 聊我们**。
- **别连续提问超过 2–3 次**（变查户口）。正解：分享自己 + 提问对方交替。
- 聊天像打拳，有出拳有收拳；**高点抽离**，在最热处主动结束，让她意犹未尽。
- **神秘感 = 让她猜**：控制频率、主动结束、回复速度与字数对等、"越忙越有魅力"。

### 拉扯/调情手法
- **推拉**：夸中带损，先肯定优点再轻否定缺点，力度绝不人身攻击。
- **双线**：直线（正经信息）+ 曲线（调侃幽默）交叉，纯直线很闷。
- **曲解**：把她的话歪解成"她在追我/占我便宜"，角色互换。
- **假定成交**：跳过询问，直接当她已答应（不说"你哪天有空"，说"哪天带你去吃那家"）。
- **否定 > 打压**：否定更淡、更显无所谓，效果反而强；打压显刻意、很 low。
- **转移注意力**：拉升话术里加一个更敏感的词，让她忽略你真正的动作。
- **意图声明/植入预期**：描述未来的亲密画面，测窗口 + 引发情绪波动。
- **自保防被拒**：预测她会拒绝就提前替她说出来（"我这么帅，你肯定不敢喜欢我"）——同样是拒绝，从你嘴里说出就成了幽默。
- **止损/切断话题**：反应变差时不恋战，主动切断既止损又降需求感。

### 万金油话题（延展性高、低风险）
1. **旅游**（最好用）：聊具体地点 + 自己的看法。四要素：人文（最显格调，要有自己观点甚至吐槽，但忌刻板印象）、风景（点出"适合拍照"）、饮食（剑走偏锋聊冷门店、打信息差）、气候。聊资不必真去过。
2. **影视**：别聊情节立意，**聊明星/近期热播**；看对象。
3. **养生**（机会最多）：生病时别说"多喝水"，要教她**缓解当下痛苦**再劝就医；专精痛经（须等她主动提）和失眠（给"有意思"的小诀窍）；减肥别推"每天跳绳 500"和减肥药。
4. **星座/MBTI**：引出冷读的契机。

### 冷读（让她觉得"你懂我"）
- 本质：打认知差——人只关注自己、忽略"别人也一样"。公式 = 分析 + 暗示。
- 拿捏：营造"我很理解你"，**绝不能**营造"我看透你了"的恐惧感。
- 可套用："你表面大大咧咧，其实心底住着个小女孩，比谁都焦虑""你看起来朋友多，但真正交心的就两三个""你太容易信任别人，又太在乎反馈，于是常有落差"。

## 3. 女性心理：识别信号

> **任何单一信号都不能单看，要多信号交叉验证**；应对核心永远是"先认可情绪/事实，再引导她表达"。

- **好感的提问信号**（她的问题=情感投资）：问你周末安排、择偶观、社交圈、"我吐槽你会烦吗"、过往情感、未来规划——分别在找共处机会、确认三观、测信任、测情绪承接、查隐患、看担当。
- **真心喜欢的显性表现**（≥2 个好感≥70%，≥4 个可推进）：主动无目的聊天、记住小众细节、展现脆弱、融入你的社交动态、允许肢体升级、主动规划未来、放下防备（素颜赴约）、分享负面情绪并要安慰。
- **肢体信号**（结合身体朝向综合判断）：身体正对、对视后害羞偷看、把玩头发、脚尖朝你、容许近距离、镜像你的动作、真心大笑（眼睛眯起）、被碰不躲甚至回碰 = 好感；侧身背对、玩手机、抱臂、刻意保持距离、僵硬躲闪 = 无兴趣。
- **隐性拒绝**（≥3 个该止损）：回复持续敷衍、拒绝所有邀约、不分享私人生活、刻意保持距离、忽视你的未来规划、频繁否定你的优点。

## 4. 框架与废物测试

### 框架
- **框架 = 互动中传递的隐含定位（谁主导、谁高价值）= 对话主导权**。
- **强框架 ≠ 态度强硬**（最常见误读）。过分强硬反成"大脾气的小男人"。框架要有**弹性**。
- 常用手法：问题抛回（她问"你喜欢什么样的"→"你觉得我会喜欢什么样的"，别正面答交出主动权）；用更大的框架罩住对方；不利时反转立场（"你该不会占够便宜想跑吧"）；犹豫时替她做决定（循序渐进地带领，不是一上来摆硬态度）。

### 废物测试应对
**四原则**：不给兴趣指标 / 不动情绪 / 不长篇解释 / 认清它是无意识的、是双刃剑。

**四句万能公式**：
1. "你上辈子也是这么说的" → 带向感性暧昧。
2. 问题反抛 →（"你是不是经常撩妹"→"你有感觉被我撩到吗"）。
3. "我是不是该假装伤心一下" → 展示框架强（应对"你不是我喜欢的类型"）。
4. 占领高地 → 点破她（"你皮这一下很开心吗"）。

**进阶（用具体事实回应）**：
- "你对每个女生都这样？"→"分人，合得来的才多花心思，我跟你聊得比别人多吧？"
- "你比我前男友差一点"→"他适合以前的你，我或许更适合现在的你。"
- "我现在不想谈恋爱"→"做朋友也好，以后想谈再考虑也不迟。"（不施压）
- "胖了你还喜欢我吗"→"喜欢的是你的幽默和热情，这些不会变，真胖了一起健身。"

## 5. 约会（线下）

- **地点**：首次避坑单独 KTV/网吧/易流汗的火锅烧烤。优先**一类地点**（自带项目、无需硬聊：溜冰场=自带牵手、宠物店=激发母性、密室/电玩/游乐场），再用**二类地点**（咖啡/清吧/公园/饭店）靠聊天升华。收尾去清吧微醺（不是灌醉）。
- **流程**：首次留大块时间（3–7 小时），**至少转场 3 次**（营造"体验多"）、**动静结合**（动态更易加深印象，也救场不擅聊天者）。
- **买单**：现实倾向男生主动，量力而行；惦记钱反而暴露扣分。
- **带闺蜜**：多半说明她今天不想推进。临时通知带→婉拒；提前说但关系一般→婉拒；聊得很好且提前说→大方让带，用"三七法则"（多照顾闺蜜引她吃醋）+ "贿赂闺蜜帮你说话"。
- **肢体接触推进**（自然 + 及时收回）：马路上扶肩靠里再松手 → 并排落座借看视频轻触观察 → 过马路先牵手腕（不是手掌）过完即松，下次不松自然滑到十指紧扣 → 夜晚人少处拥抱，对视超 2 秒主动前凑、凑到接近时停住让她完成最后动作。**只吃喝玩乐不触碰，好感不会涨**。
- **善后**：尾声当面为下一次约会铺垫（当面更易答应）；二次邀约至少隔一天，定时间别追问具体日期，直接问"你觉得哪天方便"；保持联系不失联但不跪舔。

## 6. 形象 / 展示面（硬件门槛）

- **穿搭五风格**：休闲运动 / 日系工装 / 韩系男友 / 商务霸总 / 斯文败类。六原则：不超三色、同色加打底过渡、上繁下简、配饰点缀、同色系显高级、两单品同色呼应。
- **发型**：渐层寸头（man、有安全感）、韩系碎盖、复古油头、逗号头、丸子头，按脸型/气质选。
- **个人卫生（很重要）**：修指甲鼻毛、不留胡渣、约会前必用漱口水/口香糖。减分项：指甲长、鞋脏、衣服起球、口臭。
- **朋友圈打造**（你的"把妹简历"）：四类展示——外形 / 生活品质 / 可得性（暗示单身）/ 预选（身边有异性但不暧昧）。原则：靠"藏"不靠"秀"，少而精（1–2 张），定期清理；可用"仅她可见"加深印象而不暴露需求感；评论而非点赞测窗口。
- **高能量状态**：能量会传递（你 100 她接收 50）。约会前可短暂运动调动情绪，但别耗尽。
- **态度地基**：平等、自尊（爽约放鸽子零容忍）、无需求感、提供情绪价值。

## 7. 确定关系及之后

> 详见 lifecycle.md。要点：表白要在吸引足够（暧昧充分、肢体顺畅、她给专属感）时水到渠成，而非用表白去"求"关系；很多时候自然确认比正式表白更好。之后是长期相处、情绪维护、矛盾处理、见家长、走向婚姻——靠的不再是话术，而是真诚、责任、共同成长和处理现实问题的能力。
