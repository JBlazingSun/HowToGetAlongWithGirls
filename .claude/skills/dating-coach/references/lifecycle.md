# 全周期路线图

> 🌐 **语言**: 中文（当前） · [English](./lifecycle.en.md)

供「流程 C：阶段诊断」使用。当用户问"我现在该怎么推进""我们到哪一步了""下一步干嘛"时，先**定位他在哪个阶段**，再给该阶段的任务和下一步动作。

## 阶段总览

```
阶段 0：自我建设（贯穿始终的底座）
│
认识阶段（追求期，分四个子阶段）：
   1. 吸引  →  2. 平淡  →  3. 暧昧  →  4. 恋爱
│
5. 长期相处  →  6. 走向结婚
```

**最常见的失败有两种**：① 步子迈太大（阶段没到就重度调情/表白/越界）；② 原地不推进（停在平淡期变成"好哥们/树洞"）。诊断时先定位，再判断他是"太急"还是"太怂"。

诊断方法：根据用户描述判断阶段（特征见下），指出**当前阶段的核心任务**、**他卡在哪**、**进入下一阶段的标志和动作**。结合该女孩的档案（见 profile-template.md）做个性化判断。

## 阶段 0：自我建设（贯穿始终的底座）

不是一个时间段，而是永远在做的事。任何阶段卡住，答案常常是回到这里。
- **硬件**：形象（穿搭/发型/身材/卫生）、收入与事业、生活方式。
- **软件**：情绪稳定、有趣的阅历、社交能力、不缺爱的内核。
- **社交基数**：持续认识新的人，不把全部希望压在一个人身上。
> 若用户反复在一个对象身上患得患失、需求感爆表，根因往往是选择太少 + 自我价值感低。先解决这个。

---

## 认识阶段 · 子阶段 1：吸引

- **入口（初识/破冰）**：刚加上微信、刚搭讪认识。率先打破陌生感，开场轻松（一个表情/一句调侃，别正式"你好"也别查户口），从她的情境/朋友圈找话题。
- **特征**：从"陌生/客气"过渡到"聊得起来"，但你还只是"一个还行的男生"。
- **核心任务**：从"还行"变成"让她有点在意"——传递情绪、展示价值、制造好奇。
- **该做**：推拉、双线、冷读、自然带出自己的亮点和高价值面、聊万金油话题、用神秘感和高点抽离让她惦记你；侧面展示价值（朋友圈 + 聊天）。
- **常见错误**：变成有问必答的工具人；只有信息没有情绪；跪舔讨好；连续查户口。
- **→ 进入下一阶段的标志**：她主动找你、主动开话题、回应并享受你的调侃、出现明显兴趣信号。

## 认识阶段 · 子阶段 2：平淡（最危险的瓶颈期）

- **特征**：吸引建立过了，但热度回落——聊天变成日常流水（"吃了吗/在干嘛/晚安"），有来有回却没有进展，新鲜感消退，有**滑回普通朋友**的风险。
- **为什么危险**：大多数人卡死在这里。既没勇气推进，又舍不得放手，于是把好不容易建立的吸引，用无营养的尬聊一点点磨没。
- **核心任务**：**重新制造情绪波动 + 把关系往暧昧推**，打破"平淡—变熟—变无趣"的下滑曲线。
- **该做**：
  1. **降频抽离**：别再天天找她、别秒回。主动留白，让她感到落差、想起你。"越忙越有魅力。"
  2. **注入新鲜与情绪**：换话题打法，上推拉、调侃、轻度暧昧元素、有画面感的内容，打破流水账。
  3. **侧面展示价值刷新印象**：朋友圈展示生活和高能量状态。
  4. **果断往线下推**：平淡期最好的解药是见面。用**模糊邀约**试探（"哪天带你去吃那家XX"）。
- **常见错误**：靠加大聊天频率/更殷勤来"挽回热度"（适得其反，需求感暴露）；一直等"感觉对了再约"（永远等不到）。
- **→ 进入下一阶段的标志**：她对你的推拉和暧昧有正反馈、接受邀约、线下能见面。
- **若怎么推都冷**：可能吸引本就不足或她兴趣在消退——查隐性拒绝信号，必要时降温/止损，别硬磨。

## 认识阶段 · 子阶段 3：暧昧

- **特征**：有情绪波动、会打情骂俏、容忍轻度越界、给你一些专属感、愿意线下见。
- **核心任务**：升温 + 把关系搬到线下、开始肢体接触。
- **该做**：
  1. 模糊邀约有回应后 → **假定成交**定具体时间。
  2. 线下约会按 knowledge-base 第 5 节执行（地点、转场、肢体推进）。
  3. 适度性张力、植入预期、拉升关系，力度匹配她的反馈。
- **常见错误**：永远不约线下，拖成"好哥们"；或步子太大、太猛吓退她；接不住废测。
- **→ 进入下一阶段的标志**：线下见面顺畅、肢体接触不抵触、她明显投入（规划共同未来、吃醋、给专属时间）。

## 认识阶段 · 子阶段 4：恋爱（确定关系）

- **原则**：**表白是"水到渠成的确认"，不是"用来求关系的手段"**。吸引和暧昧都很充分时，确认关系几乎是自然结果。
- **时机判断**：她持续给出兴趣信号、肢体接触顺畅、把你当"特别的人"、双方都默认在往一起走——这时确认才稳。还在使劲追、靠表白赌一把，多半早了。
- **方式**：很多情况下**自然滑入**（牵手/亲吻后顺势点破）比正式表白更好；正式仪式感适合本就八九不离十、想给她交代的情形。
- **被拒/被拖**：到这步还明显犹豫回避，说明前面吸引不够——别反复逼问，退回做吸引或重新评估，必要时止损。

## 阶段 5：长期相处

话术退场，**真东西登场**：
- **维护情绪浓度**：别一确定关系就懈怠变回无趣。继续约会、制造体验、表达欣赏。
- **情绪共鸣**：她倾诉要的是被懂，不是被纠正。先共情再谈事。
- **矛盾处理**：先讲感受（"我有点委屈"）再讲问题，别指责、翻旧账、冷战。冷战是关系杀手。
- **边界与尊重**：不跪舔也不打压，平等，保留各自空间和社交。
- **观察是否合适长期**：情绪是否双向承接、吵架是否愿沟通、是否愿把你纳入她的未来和社交圈。
- **现实推进**：见朋友 → 见家长 → 谈未来规划（城市、买房、婚姻节奏）。

## 阶段 6：走向结婚

- **核心已不是吸引，而是匹配与经营**：三观、家庭背景、金钱观、生活习惯、未来规划是否合得来。
- **关键事项**：见双方家长、坦诚谈彩礼/买房/两地/要不要孩子等现实问题、共同处理矛盾的能力、对长期承诺的真诚。
- **求婚**：建立在双方都已确认要走下去的基础上，仪式是表达，不是说服。
- **底线提醒**：到这步，诚实、责任感、共同成长远比技巧重要。若需要靠"技巧"维持，关系本身可能有问题——帮用户看清，也是教练的职责。

## 快速诊断速查

| 用户的处境 | 大概率所在阶段 | 第一步该做 |
|---|---|---|
| "刚要到微信不知道聊啥" | 吸引（初识） | 轻破冰、找共同点、别查户口 |
| "聊得挺好但没火花/感觉像朋友" | 吸引未稳 或 已滑入平淡 | 加情绪和推拉，制造好奇 |
| "以前聊得火热，现在变成日常尬聊/没话题" | **平淡期** | 降频抽离 + 注入新鲜 + 果断模糊邀约推线下 |
| "她爱回不回了" | 平淡下滑 或 隐性拒绝 | 查信号：降温控需求感，或止损 |
| "想约她出来" | 暧昧 | 模糊邀约 → 假定成交，看 knowledge-base 第 5 节 |
| "约会了，下一步？" | 暧昧后期 | 推进肢体、铺垫下次、评估能否确认 |
| "想表白/确定关系" | 准恋爱 | 先判断吸引是否足够，能则自然确认，不能则退回做吸引 |
| "在一起了但常吵架/冷淡" | 长期 | 情绪维护 + 矛盾沟通法，回到阶段 5 |
| "考虑结婚但犹豫" | 走向结婚 | 谈现实匹配，不是谈技巧 |
